Learning/Practice Project

Puzzle

This project marked my introduction to game development using Unity. Created in late 2021, it showcases my initial exploration of game mechanics, design, and interactivity.
The game is a puzzle-solving experience where players must clear the board strategically. Each piece moves in the direction indicated by an arrow, requiring careful planning to access inner pieces by first moving the outer ones. Adding to the challenge, the game features turn limits and 15 handcrafted puzzle levels, all designed by me. This project represents the foundation of my skills in Unity and puzzle design, combining creativity and logic to craft an engaging and rewarding gameplay experience.

Fish Adventure

Fish Adventure is a thrilling survival game where players must protect a fish from incoming threats like sharks and dangerous fishes attacking from all four directions within a set time. Players can use strategic power-ups, such as slowing obstacles, reducing survival time, and eliminating half of the threats, to gain an edge. The game features a shop system for purchasing skins and items with coins or through in-app purchases, enhancing customization. Unity Ads integration offers rewards for watching ads, adding extra engagement. With its exciting gameplay, power-ups, and customization options, Fish Adventure provides a captivating and enjoyable experience.

Touch 'n' Play Wonderland

This project was a collection of interactive mini-games designed for children, tailored for Windows touch panels in a playland setting. The games included fun and engaging activities like Fruit Pop, football kicking, leaf blowing, smiley hunting, and star or heart generation wherever the screen was touched. Each game encouraged playful interaction, offering instant visual and tactile feedback as children walked on or touched the panels. Designed to captivate and entertain, this project transformed simple touch interactions into a delightful and immersive experience for young audiences.
